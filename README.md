# Pug

Pug is a template engine for Node and for the browser. 

## Installation

Use the package manager [guide](https://www.sitepoint.com/a-beginners-guide-to-pug/) to install pug.

```bash
   sudo npm install pug-cli -g
   npx pug --v
```

## Usage

```pug

  pug -w ./ -o ./html -P


```
